# [P研] 倉庫スケジュール — 引き継ぎ資料

## このサイトについて

P研の倉庫利用スケジュールを管理するための Web アプリです。
URL を開くだけで使えるシンプルな設計になっており、インストールや会員登録は不要です。
スマホ・PC どちらからでも同じ URL でアクセスでき、iOS のホーム画面に追加するとアプリのように使えます。

---

## 使い方（一般ユーザー）

### 予定を確認する

画面にアクセスするとその週（火曜〜月曜）のスケジュールがカレンダー形式で表示されます。
予定ブロックをタップ（クリック）すると、日付・時間帯・所要時間の詳細が確認できます。

### 予定を追加する

1. 「＋ 予定を追加」ボタンを押す
2. 名前・曜日・開始〜終了時刻・4桁の PIN を入力する
3. 「追加する」を押すと全員のカレンダーに反映される

複数の予定をまとめて登録したい場合は「＋ 行を追加」で行を増やせます。
行が 2 行以上になると「PIN 一括設定」欄が表示され、全行に同じ PIN をまとめて設定できます。

> **PIN について**
> 予定を登録するときに自分で決める 4 桁の数字です。
> 後で自分の予定を編集・削除するときに入力を求められるので、忘れないようにしてください。
> 他の人の PIN は分からない仕組みになっています。

### 予定を編集・削除する

| 操作 | 方法 |
|------|------|
| スマホ | ブロックをタップ → ポップアップ内の「編集」または「削除」ボタン |
| PC | ブロックを右クリック → コンテキストメニューから操作 |

編集・削除には登録時に設定した PIN の入力が必要です。
